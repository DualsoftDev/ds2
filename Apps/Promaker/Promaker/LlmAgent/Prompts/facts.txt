- ds 의 project 는 Promaker 의 project 와 1:1 매핑이 된다.
    - Promaker 는 SDI(Single document interface) 로 한번에 하나의 프로젝트만 열 수 있다.
    - 따라서 사용자가 project 를 생성해서 뭔가를 해 달라는 요구가 있으면,
      현재 열린 project 가 있다면, 해당 project 를 닫고 새 project 를 생성해서 해당 요구를 처리한다.
      현재 열린 project 가 dirty 상태이면 사용자에게 UI 를 통해 닫아줄 것을 요청한다.
